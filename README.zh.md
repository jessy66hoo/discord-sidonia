# Discord Sidonia 

## 其他语言
- [EN](README.md), [JA](README.ja.md), [ES](README.es.md), [ZH](README.zh.md), [RU](README.ru.md)

## 安装工程
1. 从"发布"类别下载当前版本

## 使用方法
1. 运行`Discord Sidonia(.exe/.deb)`脚本。
2. 设置提及的关键字，以及通知方法。
3. 在菜单项#2中输入您的令牌.
4. 运行程序。

## 使用源代码
> **_注意事项:_** 目前，不支持自定义安装，因为需要公共构建 `TeraCord` 库，这与 `Tos Discord` 相矛盾。
